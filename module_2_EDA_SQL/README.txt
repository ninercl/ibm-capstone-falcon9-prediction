Este módulo forma parte del proyecto final del **IBM Data Science Professional Certificate**. En esta etapa, realizamos un Análisis Exploratorio de Datos (EDA) utilizando SQL sobre una base de datos de lanzamientos de SpaceX.

---

## 📘 Descripción

En este módulo trabajamos directamente en los **Skills Network Labs** de IBM, un entorno virtual preparado para correr notebooks Jupyter con acceso a SQL y SQLite.

Se realiza un análisis exploratorio sobre datos relacionados con lanzamientos de cohetes **Falcon 9** de SpaceX, utilizando exclusivamente comandos SQL para:

- Consultas básicas y complejas
- Filtros y agregaciones
- Agrupaciones (`GROUP BY`)
- Joins entre tablas
- Detección de outliers y patrones iniciales
